# Jinzheng Ren — Jekyll Academic Website

This site is a GitHub Pages / Jekyll version of the single-file academic website.

## Files to update later

- `_data/profile.yml` → biography, interests, education, profile links
- `_data/experience.yml` → research positions
- `_data/projects.yml` → selected projects
- `_data/publications.yml` → selected publications
- `_data/contact.yml` → contact details
- `assets/images/avatar.png` → profile photo
- `assets/files/JinzhengRen_CV_2026_01_03_final.pdf` → latest CV PDF

## How to publish on GitHub Pages

1. Create a GitHub repository (for example `yourusername.github.io` or a project repo).
2. Upload all files from this folder.
3. In the repository settings, enable **GitHub Pages** from the default branch.
4. GitHub Pages will build the Jekyll site automatically.
